# ROSCA Web3 DApp

Blockchain-based community savings and lending platform inspired by Bloinx.

## Tech Stack

### Frontend
- Next.js
- TailwindCSS
- wagmi
- RainbowKit

### Smart Contract
- Solidity
- Hardhat
- OpenZeppelin

### Backend
- Node.js
- Express
- MongoDB

### Storage
- IPFS / Pinata

### Network
- Sepolia Testnet
